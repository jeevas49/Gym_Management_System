# Gym Management System

Welcome to the Gym Management System! This C++ program allows you to manage gym memberships efficiently. You can create new memberships, view existing memberships, modify memberships, cancel memberships, and search for memberships based on member names.

## Features

- **Create a New Membership**: Enter the member's name and select the timing (morning, afternoon, or evening) to create a new membership.
- **View Existing Memberships**: View all existing memberships sorted by timing.
- **Modify Membership**: Modify a membership by selecting its number and entering a new name and timing.
- **Cancel Membership**: Cancel a membership by selecting its number.
- **Search for Membership**: Search for a membership by entering the member's name.

## Usage

1. Compile the program using a C++ compiler.
2. Run the compiled executable.
3. Follow the on-screen menu to perform various operations.

## Menu Options

- **1. Create a new membership**: Enter 1 to create a new gym membership.
- **2. View existing memberships**: Enter 2 to view all existing gym memberships.
- **3. Modify a membership**: Enter 3 to modify an existing gym membership.
- **4. Cancel a membership**: Enter 4 to cancel an existing gym membership.
- **5. Search for a membership**: Enter 5 to search for a gym membership by name.
- **6. Exit**: Enter 6 to exit the program.

## Note

- Timing options: 1 for morning, 2 for afternoon, 3 for evening.

Thank you for using the Gym Management System!
